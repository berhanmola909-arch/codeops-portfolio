five Terminal commands:
 1,mv: To move folder and files also it can be use to rename them
 2,cd: To change directory
 3,touch: To create a file
 4,mkdir: To create a folder
 5,clear: To clear the directory

3 git commands:

1,git init: To initialise a new local repo
2,git add: To move changes from working directory to staging area
3,git commit: To save changes
